Independent study : robotical arm 機械手臂

Introduction :

這幾年人工智慧越來越盛行，有關下棋的手臂最初起源於AlphaGo，然而近幾年AlphaGo已悄悄淡出，最新推出的AlphaGo Zero可以說是AlphaGo的進化版。想把 AlphaGo Zero 變成更通用化、能下更多不同棋的演算法時，就需要重新思考處理方法。

Design concept : 

老年化的問題日益嚴重，許多家庭常有老人獨自在家電視一開就是一整天，造成老年癡呆的問題日益逼近，最大的問題就是沒有一個''伴''，老人們都會下棋，但因行動不便可能無法出門，所以設計一個下棋手臂，讓老人們在家除了看電視可以有更好的娛樂項目，如此一來也能藉由下棋來降低老年癡呆的速度。

Issue ＆ Solution

Q1

◎問題

組員 git push 後，另外其他五人在近端(localhost:8443)上看不到組員成功上傳的資料。

◎解決辦法

在start.bat上，先進入倉儲資料夾，接著要先 '' git stash '' 再 '' git pull '' ，重新整理後即可看到完整頁面。

Q2

◎問題

成功推送上去後重新整理頁面發現沒有東西，但近端顯示是有的，在commit上也顯示推送成功

◎解決辦法

因網頁間是互相串連，所以要先重新整理再載入，有時會因為''proxy''限制，導致網頁改版速度變慢，通常都是幾個小時改版一次，可是因為我們正在設計，所以有很大的可能一分鐘更改好幾次，故網頁改版速度遠不及我們重新整理的速度，所以需要時間等待再重新載入，若還是沒出現，就要去程式碼看是否有錯誤
