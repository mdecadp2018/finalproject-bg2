課程內容

專題進度

W10

◎進度

1.下載新版p37

2.更改小組的  README.md

3.更改網站簡介及blog

4.整合各組員的Onshape

◎組員分工

每位組員先思考想做的專題題目，下週決定出bg2的專題題目

W11

◎進度

決定好專題題目—機械手臂

◎組員分工

每位組員先回家自行看些機械手臂設計之文獻，下週確定好要做的機械手臂類型

疑難排解

Q1

◎問題

組員 git push 後，另外其他五人在近端(localhost:8443)上看不到組員成功上傳的資料

◎解決辦法

在start.bat上，先進入倉儲資料夾，接著要先 '' git stash '' 再 '' git pull '' ，重新整理後即可看到完整頁面。
